Welcome to Tallinux. This is a custom Linux distribution inspired by the city of Tallinn. Tallinn has an old medieval town but also very fast internet everywhere. I wanted this system to be exactly like that. It is very stable but also fast.

## Features
* Fast boot: It starts up quickly.
* No bloatware: Just the essential programs. No useless background applications.
* Stable base: It does not crash easily when you are working.
* Simple design.

## How to Install
It is simple to set up.
1. Download the ISO file from the releases page.
2. Flash it to a USB drive with a tool like Rufus or dd.
3. Boot from the USB and follow the prompts on the screen.
